# PhysEngine

PhysEngine is a physics engine for rigid body animation.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

Prerequisites for [CMake](https://cmake.org/).

### Installing on Windows

1. clone the repo

```
git clone
```

2. Run CMake-GUI, configure and generate the project

3. Open the project with Visual Studio

4. Compile

### Installing on Linux

```
mkdir build
cmake ../
make -j4
```

## Running the tests

The testcases are in `./examples`.
